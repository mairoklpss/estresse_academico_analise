# 🤓 Dashboard de Índice de Estresse Acadêmico

## 📋 Visão Geral

Este projeto é um dashboard interativo desenvolvido em **Streamlit** para análise e visualização dos níveis de estresse acadêmico entre estudantes. O dashboard permite explorar como diferentes fatores (ambiente de estudo, pressões sociais, estratégias de enfrentamento, etc.) influenciam o estresse dos alunos.

## 🚀 Funcionalidades


### 📈 Visualizações Disponíveis

| Gráfico | Descrição |
|---------|-----------|
| Distribuição de Estresse | Visão geral dos níveis de estresse entre todos os alunos |
| Análise por Ambiente | Comparação do estresse em ambientes: Tranquilo, Barulhento e Inquieto |
| Análise por Estágio Acadêmico | Distribuição do estresse entre Graduação, Ensino Médio e Pós-graduação |
| Análise por Competição | Relação entre competição acadêmica e níveis de estresse |
| Perfil de Estresse Alto | Identificação dos grupos mais afetados por estresse elevado |

### 🔍 Filtros Interativos

- Nível de Estresse Acadêmico
- Estágio Acadêmico
- Pressão dos Colegas
- Pressão Acadêmica Familiar
- Ambiente de Estudo
- Competição Acadêmica
- Estratégia de Enfrentamento
- Hábitos Diários (Fumo/Álcool)

## 🛠️ Tecnologias Utilizadas

- **Google Colab**: Ferramenta usada para análise do DataFrame
- **Python**: Linguagem principal
- **Streamlit**: Framework para criação do dashboard
- **Pandas**: Manipulação e tratamento dos dados
- **Plotly Express**: Criação de gráficos interativos
